principais formatações

negrito/destaque

vc pode colocar em engrito usando a tag <b> e </b> por exemplo

nessa frase aqui usamos a sigla strong <strong>

italico/enfase

aqui vamos utilizar de italico <i>

aqui vc pode ver a utilização do enfase :D <em>

texto marcado

podemos criar um texto marcado com a tag mark <mark>

Texto grande e pequeno

este aqui é um texto pequeno <small> e esta aqui é grande

Texto deletado

Podemos marcar um texto como excluido <del> , para indicar que ele deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido <ins>, para dar uma enfase e indicar que foi adicionado, tambem podemos usar para subliar o mesmo <u>

Texto subrescrito

para inserir coisas do tipo x20+3, x 20+3 <sup>

Texto subscrito

para inserir coisas como h2o, h 2o <sub>
